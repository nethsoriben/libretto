# Libretto
A terminal-based music library organizer with metadata tagging and dynamic, rule-based playlists. Browse and filter 15+ tracks live using vim-like keyboard controls.


A console TUI for organizing music libraries through structured metadata tags and interactive keyboard-driven browsing.

Tag tracks with genre, mood, energy, and custom fields. Apply filters that update the view instantly. Save filter combinations as dynamic playlists that auto-include new matching tracks.

Built with clean separation between metadata, filtering logic, and display layers. Extensible by design — add new tags without modifying the core engine.
